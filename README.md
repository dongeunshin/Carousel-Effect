# Carousel-Effect

회전 목마처럼 cell들이 돌고 있는 거 처럼 보이는 것을 Carousel-Effect라고 부른다.<br>
이러한 효과를 collectionview에 적용한다면 좀 더 다채로운 애니메이션을 보여줄 수 있다!<br>
